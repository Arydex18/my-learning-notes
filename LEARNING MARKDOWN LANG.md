# 25. Markdown Rules

✔ One blank line between sections

✔ Use headings

✔ Use bullet lists

✔ Keep paragraphs short

✔ Use code blocks for commands

✔ Explain concepts in your own words

✔ Add examples

✔ End every note with Key Takeaways

---


That's enough for almost every note you'll ever write.
