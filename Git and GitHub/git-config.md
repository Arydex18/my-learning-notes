# Git Clone

## 📖 Source

- Course: Apna College Git & GitHub
- Lecture: Git Clone

---

## 🎯 Goal

Learn how to copy a GitHub repository to my computer.

---

## 🧠 What is it?

`git clone` downloads a complete copy of a GitHub repository onto my local computer.

---

## 💻 Command

```bash
git clone https://github.com/Arydex18/my-learning-notes.git
```

### What it does

Creates a local copy of the repository.

---

## 📝 Example

```bash
git clone https://github.com/Arydex18/my-learning-notes.git
```

After running it, a folder named `my-learning-notes` appears on my computer.

---

## 💡 My Understanding

GitHub stores my project online.

`git clone` is like downloading that project onto my laptop so I can work on it.

---

## ⚠️ Things I Keep Forgetting

- Clone is done only once.
- After cloning, I work inside the cloned folder.
- `git push` uploads changes back to GitHub.

---

## 🔑 Key Takeaways

- `git clone` downloads a repository.
- It creates a folder.
- I only need to clone once.
